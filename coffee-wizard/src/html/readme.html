What is Black Coffee?

Black Coffee provides an infrastructure for building and maintaining front end and back end automated tests. It is designed to match the functionality of Selenium (front end) and various URL Libraries (back end).

System Requirements:

This wizard contains a Universal installer for Mac OS X 10.9.x and requires Python 2.7+ and JVM 1.6+ installations to function - all present by default on Mac OS X Lion+.

Install Location:

The Black Coffee wizard copies Black Coffee to the directory you target.

What is Installed:

Most Black Coffee files are installed to your directory. Black Coffee is a bundled jar running on your JVM and Jython. Other files installed include python dependencies such as urllib and WebDriver.

Documentation:

The BlackCoffee Wiki is available at: <> and should be consulted for further documentation and support. For further info, please see selenium.org and https://docs.python.org/2/library/urllib.html .
