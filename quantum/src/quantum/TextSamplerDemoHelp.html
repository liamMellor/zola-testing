In standard HTML formatting, a tag would be represented as so:

<a href='/dummyURL' id='thisUrl' class='thisClass'> Click Here! </a>

'a' refers to an anchor tag. This is a clickable button (generally)

href is a hypertext reference. The characters following are generally the link to a URL or web address

id is the (hopefully) unique identifier for the tag. This will be your go-to identifier for creating these Selenium scripts.

class is the CSS class. this is also targetable by selenium in the event that an id is not available

'Click Here!' is the inner HTML of the tag. This is what is displayed to the user.

"" is the closing tag. This tells the HTML interpreter (aka a webpage) that the tag is done.

There are many tag types in HTML. All of these tags have some kind of unique identifer. It is your job to find this unique identifiers.
